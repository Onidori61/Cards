Projeto pessoal criado para testar Tailwind CSS e outras ideias de layout e interação.

O objetivo é desenvolver uma lista de cards no formato de “diários”, onde cada card contém um título, um texto e uma assinatura, acompanhados de uma imagem de fundo. Os cards são responsivos e utilizam efeitos visuais como blur de fundo e sobreposição para manter a legibilidade do conteúdo.

O projeto é usado principalmente para escrever e organizar textos pessoais destinados à minha namorada, explorando diferentes estilos visuais e animações.
